# 内容管理

我们将内容管理分为四个模块：

#### 消息管理

消息指的是机器人和用户交互过程中推送的所有内容卡片、动作卡片等

#### 图文管理

图文是我们在消息按钮中进入的H5文章内容

#### 图片管理

管理了我们所有上传过的图片

#### 视频管理

管理了我们所有上传过的视频
