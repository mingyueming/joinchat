# 智能回复

在JoinChat的使用过程中，我们把用户可以主动触发机器人预置消息的入口全部放在“智能回复”功能模块中，该模块也是机器人对话交流的配置核心。如何将这些入口消息配置的更加科学、有趣，则决定了用户对机器人的好感度和用户粘度。
