# 授权登陆

首先需要登陆我们的官网：[ https://www.joinchat.ai](%20https://www.joinchat.ai)，如下图：

![JoinChat&#x9996;&#x9875;](../.gitbook/assets/image%20%2897%29.png)

{% endhint %}

## 使用Facebook登陆

如果您的浏览器没有登陆Facebook账号，则需要先登陆您的账号。
